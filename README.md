# Twitter Analytics Dashboard 📊

This project is a Twitter Analytics Dashboard created using Power BI.

## 📌 Overview
This dashboard provides insights into:
- Tweet performance by week
- Media engagement vs media views
- Tweet count trends
- URL clicks analysis
- Engagement rate and impressions

## 🛠 Tools Used
- Power BI
- Excel (Data Cleaning & Preparation)

## 📊 Key Metrics
- Total Tweets: 1181
- Likes: 8187
- Media Views: 89K
- Engagement Rate: 0.07

## 🚀 Purpose
This project demonstrates data visualization and analytical skills using real-world styled social media data.
